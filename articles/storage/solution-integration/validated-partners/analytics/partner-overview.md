# <a name="azure-storage-analytics-and-big-data-partners"></a>Azure Storage Analytics en big data partners

In dit artikel worden micro soft-partner bedrijven uitgelegd die zijn geïntegreerd met Azure Data Lake Storage. Deze partner oplossingen beslaan werk belastingen zoals moderne Data Warehouse-workloads, geavanceerde analyses en real-time analyses. Deze partners profiteren van de [hiërarchische naam ruimte](../../../blobs/data-lake-storage-namespace.md) in azure Storage om de oplossing te optimaliseren en efficiënt uit te voeren in Azure.
