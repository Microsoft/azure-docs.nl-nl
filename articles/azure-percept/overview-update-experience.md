# <a name="azure-percept-dk-update-experience-overview"></a>Overzicht van Azure percept DK update-ervaring

Met Azure percept DK kunt u het besturings systeem van de dev kit en de firmware over-the-Air (OTA) of via USB bijwerken. OTA update is een eenvoudige manier om apparaten up-to-date te houden via de [Update van het apparaat voor IOT hub](https://docs.microsoft.com/azure/iot-hub-device-update/) service. USB-updates zijn beschikbaar voor gebruikers die geen OTA-updates kunnen gebruiken of wanneer het apparaat opnieuw moet worden ingesteld. Bekijk de volgende instructies om aan de slag te gaan met de updates voor Azure percept DK-apparaten:

- [Azure IoT Hub instellen om over-the-Air (OTA)-updates te implementeren in uw Azure percept DK](./how-to-set-up-over-the-air-updates.md)
- [Uw Azure percept DK over-the-Air bijwerken (OTA)](./how-to-update-over-the-air.md)
- [Uw Azure percept DK via USB bijwerken](./how-to-update-via-usb.md)
