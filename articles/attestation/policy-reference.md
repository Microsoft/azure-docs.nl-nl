# <a name="azure-policy-built-in-definitions-for-azure-attestation"></a>Azure Policy ingebouwde definities voor Azure Attestation

Deze pagina is een index van [Azure Policy](../governance/policy/overview.md) ingebouwde beleidsdefinities voor Azure Attestation. Zie [Ingebouwde Azure Policy-definities](../governance/policy/samples/built-in-policies.md) voor aanvullende ingebouwde modules voor Azure Policy voor andere services.
