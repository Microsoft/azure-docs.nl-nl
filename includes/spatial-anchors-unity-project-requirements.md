De Azure spatiale-ankers SDK moet worden geconfigureerd in <a href="https://unity.com/releases/2019-lts" target="_blank">unit 2019,4 (LTS)</a> met behulp van de volgende pakketten:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- ARCore XR-invoeg toepassing (voor ondersteuning van Android-apparaten): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR-invoeg toepassing (voor iOS-apparaat-ondersteuning): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> De ASA-SDK ondersteunt momenteel het <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">Framework unit XR-invoeg toepassing</a> niet
