# <a name="azure-active-directory-b2c-isv-partners"></a>Azure Active Directory B2C ISV-partners

Ons ISV-partnernetwerk breidt onze oplossingsmogelijkheden uit om u te helpen naadloze ervaringen voor eindgebruikers te bouwen. Met Azure AD B2C kunt u integreren met ISV-partners om MFA-methoden (Multi-Factor Authentication) in te schakelen, op rollen gebaseerd toegangsbeheer uit te voeren, identiteitsverificatie en -controle in te schakelen, beveiliging met botdetectie en fraudebeveiliging te verbeteren en te voldoen aan de vereisten van Payment Services Directive 2 (PSD2) Secure Customer Authentication (SCA). Gebruik ons gedetailleerde voorbeeld voor meer informatie over het integreren van apps met de ISV-partners.

>[!NOTE]
>De [Azure Active Directory B2C communitysite op GitHub biedt](https://azure-ad-b2c.github.io/azureadb2ccommunity.io/) ook aangepaste voorbeeldbeleidsregels van de community.
