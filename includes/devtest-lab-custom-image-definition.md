In Azure DevTest Labs kunt u met aangepaste installatie kopieën snel Vm's maken zonder te wachten totdat alle vereiste software op de doel computer is geïnstalleerd. Met aangepaste installatie kopieën kunt u alle software die u nodig hebt in een VHD-bestand vooraf installeren en vervolgens het VHD-bestand gebruiken om een virtuele machine te maken. Omdat de software al is geïnstalleerd, is de aanmaak tijd van de virtuele machine veel sneller. Daarnaast kunnen aangepaste installatie kopieën worden gebruikt om Vm's te klonen door een aangepaste installatie kopie van een virtuele machine te maken en vervolgens Vm's te maken op basis van die aangepaste installatie kopie.
