# <a name="legacy-devices-on-the-azure-certified-device-catalog"></a>Verouderde apparaten in de Azure Certified-stuurprogrammacatalogus

Mogelijk hebt u in de Azure-gecertificeerde apparaatinstantie gezien dat sommige apparaten geen blauw overzicht of het label ' Azure-gecertificeerd apparaat ' hebben. Deze apparaten (' verouderde apparaten ') zijn eerder gecertificeerd onder het oudere programma.

## <a name="certified-for-azure-iot-program"></a>Gecertificeerd voor Azure IoT-programma

Voordat het Azure Certified Device-programma wordt gestart, kunnen de hardware-partners hun producten eerder certificeren onder het programma Certified for Azure IoT. Het Azure Certified Device-certificerings programma richt zich op de missie van klanten in plaats van de mogelijkheden van technische apparaten.

Apparaten die zijn gecertificeerd als een ' IoT Hub gecertificeerd apparaat ', worden weer gegeven in de Azure Certified-apparaatinstantie als een ' verouderd apparaat '. Dit label geeft aan dat apparaten eerder zijn gekwalificeerd via het nu buiten gebruik gestelde programma, maar niet zijn gecertificeerd via het bijgewerkte Azure Certified Device-programma. Deze apparaten worden duidelijk genoteerd in de catalogus door hun gebrek aan blauwe omtrek en kunnen worden gevonden via het filter ' IoT Hub Certified devices (verouderd) '.
