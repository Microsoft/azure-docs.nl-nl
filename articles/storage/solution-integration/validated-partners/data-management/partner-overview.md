# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Azure Storage Data Governance-, beheer-en migratie partners

In dit artikel wordt uitgelegd hoe micro soft-partner bedrijven zijn geïntegreerd met Azure Storage die de algehele mogelijkheden voor gegevens beheer van klanten verbeteren. Deze partner oplossingen kunnen ondersteuning bieden voor opslag analyse en rapportage, neutraal migratie, replicatie, Cloud lagen en data governance.
