# <a name="quickstart-create-a-cognitive-services-resource-using-the-azure-management-client-library"></a>Quickstart: Een Cognitive Services-resource maken met behulp van de Azure Management-clientbibliotheek

Gebruik deze quickstart om Cognitive Services-resources te maken en beheren met behulp van de Azure Management-clientbibliotheek.

Azure Cognitive Services is een reeks Cloud-basis Services met REST Api's en client bibliotheken die ontwikkel aars kunnen helpen bij het bouwen van cognitieve intelligentie in hun toepassingen. Ontwikkel aars hebben geen directe, kunst matige intelligentie (AI) of vaardig heden met gegevens wetenschap of kennis nodig om succes te behalen. Met Azure Cognitive Services kunnen ontwikkelaars eenvoudig cognitieve functies toevoegen aan hun toepassingen met cognitieve oplossingen die kunnen zien, horen, spreken en begrijpen. Er zijn zelfs al toepassingen die beginnen te redeneren.

Afzonderlijke AI-services worden vertegenwoordigd door Azure[-resources](../azure-resource-manager/management/manage-resources-portal.md) die u onder uw Azure-abonnement hebt gemaakt.
