Maak verbinding met uw Windows-VM met behulp van de Remote Desktop Protocol (RDP) via het IP-adres dat u hebt door gegeven tijdens het maken van de virtuele machine.

1. Start RDP op uw client. Ga naar **Start** en typ **Mstsc**.
1. Voer het IP-adres van de virtuele machine in en geef de toegangs referenties op die u hebt gebruikt in het parameter bestand voor de VM-sjabloon.

    ![Scherm afbeelding die laat zien hoe u via RDP verbinding maakt met uw Windows-VM.](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)
1. Mogelijk moet u de verbinding met een niet-vertrouwde machine goed keuren. 
1. U bent nu aangemeld bij uw virtuele machine die wordt uitgevoerd op het apparaat.
