# <a name="azure-policy-regulatory-compliance-controls-for-azure-spring-cloud"></a>Azure Policy regulerende nalevings controles voor Azure lente-Cloud

[Naleving van regelgeving in Azure Policy](../governance/policy/concepts/regulatory-compliance.md) biedt door Microsoft gemaakte en beheerde initiatiefdefinities, bekend als _ingebouwde modules_, voor de **nalevingsdomeinen** en **beveiligingsmaatregelen** met betrekking tot verschillende nalevingsstandaarden. Op deze pagina vindt u de **compatibiliteits domeinen** en **beveiligings controles** voor Azure lente-Cloud. U kunt de ingebouwde modules voor een **beveiligingsmaatregel** afzonderlijk toewijzen om uw Azure-resources de specifieke standaard te laten naleven.
