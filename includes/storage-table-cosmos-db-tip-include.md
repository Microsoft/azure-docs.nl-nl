> [!TIP]
> De inhoud in dit artikel is van toepassing op de oorspronkelijke versie van Azure Table Storage. Er is nu echter een alternatief voor tabelopslag: de Azure Cosmos DB Table-API. Deze API biedt hogere prestaties en beschikbaarheid, globale distributie en automatische secundaire indexen. Deze is ook beschikbaar in een [serverloze](../articles/cosmos-db/serverless.md) modus op basis van verbruik. Er zijn enkele [functieverschillen](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage) tussen de Table-API in Azure Cosmos DB en Azure-tabelopslag. Zie voor meer informatie [Azure Cosmos DB Table-API](../articles/cosmos-db/table-introduction.md).
