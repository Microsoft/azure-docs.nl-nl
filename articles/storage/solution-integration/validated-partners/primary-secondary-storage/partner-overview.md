# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Azure Storage partners voor de primaire en secundaire opslag

Dit artikel bevat een overzicht van micro soft-partner bedrijven die een NAS-of SAN-oplossing leveren, on-premises, in azure of hybride oplossing die gebruikmaakt van Azure Storage als een kosten efficiënte laag. Deze oplossingen kunnen klanten in staat stellen om dezelfde oplossing te gebruiken in een van hun omgevingen.
