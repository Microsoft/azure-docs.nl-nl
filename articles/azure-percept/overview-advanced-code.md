# <a name="advanced-development-with-azure-percept"></a>Geavanceerde ontwikkeling met Azure percept
Met Azure percept kunnen software ontwikkelaars en gegevens wetenschappers gebruikmaken van geavanceerde code werk stromen voor het beheer van de AI-levens cyclus. Dankzij een groeiende open-source bibliotheek kunnen ze voor beelden gebruiken om aan de slag te gaan met hun AI-ontwikkelings reis en productie klare oplossingen te bouwen.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Aan de slag met de zelf studies voor geavanceerde ontwikkel aars

Deze [Jupyter-notebook](https://github.com/microsoft/Project-Santa-Cruz-Private-Preview/blob/main/Sample-Scripts-and-Notebooks/Official/Machine%20Learning%20Notebooks/Transferlearningusing_SSDLiteV2%20Model.ipynb) voert overdrachts training uit met behulp van een vooraf getraind tensor flow model (MobileNetSSDV2Lite) op AzureML in python met een aangepaste gegevensset om Bowlen te detecteren. Er zijn twee zelfstudie opties die u helpen bij het werken met het notitie blok voor het trainen en implementeren van uw Bowl detector voor het project Santa Cruz Devkit:

- [Cloud ontwikkeling](./advanced-development-cloud.md): in deze zelf studie voert u het notitie blok in de [Azure machine learning portal](https://ml.azure.com) uit met een extern reken exemplaar.

- [Lokale ontwikkeling](./advanced-development-local.md): in deze zelf studie voert u het notitie blok lokaal uit in VS code met een extern reken proces.
