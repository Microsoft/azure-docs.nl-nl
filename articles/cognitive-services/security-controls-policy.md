# <a name="azure-policy-regulatory-compliance-controls-for-azure-cognitive-services"></a>Azure Policy controles voor naleving van regelgeving voor Azure Cognitive Services

[Naleving van regelgeving in Azure Policy](../governance/policy/concepts/regulatory-compliance.md) biedt door Microsoft gemaakte en beheerde initiatiefdefinities, bekend als _ingebouwde modules_, voor de **nalevingsdomeinen** en **beveiligingsmaatregelen** met betrekking tot verschillende nalevingsstandaarden. Op deze pagina worden **de nalevingsdomeinen en** **beveiligingsmaatregelen voor Azure Cognitive Services.** U kunt de ingebouwde modules voor een **beveiligingsmaatregel** afzonderlijk toewijzen om uw Azure-resources de specifieke standaard te laten naleven.
