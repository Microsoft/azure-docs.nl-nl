# <a name="supported-browsers"></a>Ondersteunde browsers 

Azure controle sfeer liggen ondersteunt de volgende browsers. U wordt aangeraden om de meest recente browser te gebruiken die compatibel is met het besturingssysteem. 

* Microsoft Edge (meest recente versie)
* Safari (meest recente versie, alleen Mac)
* Chrome (meest recente versie)
* Firefox (meest recente versie)

## <a name="chrome-incognito-mode"></a>Chrome Incognito-modus

 Chrome Incognito die cookies van derden blok keren, moeten worden uitgeschakeld voor een goede werking van controle sfeer liggen Studio.
