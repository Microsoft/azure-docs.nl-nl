# <a name="deprecation-of-azure-service-manager-support-for-azure-service-bus-relay-and-event-hubs"></a>Afschaffing van Azure Service Manager ondersteuning voor Azure Service Bus, Relay en Event Hubs

Resource Manager, onze nieuwe cloudinfrastructuurstack, wordt het 'klassieke' Azure Service Management-model (klassiek implementatiemodel) volledig vervangen. Als gevolg hiervan worden REST API's voor het klassieke implementatiemodel en ondersteuning voor Service Bus, Relay en Event Hubs op 1 november 2021 ingetrokken. Deze afschaffing werd voor het eerst aangekondigd op een aankondiging van de [Microsoft Tech Community,](https://techcommunity.microsoft.com/t5/Service-Bus-blog/Deprecating-Service-Management-support-for-Azure-Service-Bus/ba-p/370909)maar we hebben onlangs besloten om de afschaffingsperiode twee jaar langer uit te breiden vanaf het moment van de oorspronkelijke aankondiging. Voor eenvoudige identificatie hebben deze API's `management.core.windows.net` hun URI. Raadpleeg de volgende tabel voor een lijst met de afgeschafte API's en de Azure Resource Manager API-versie die u nu moet gebruiken.

Als u Service Bus, Relay en Event Hubs wilt blijven gebruiken, gaat u naar Resource Manager 31 oktober 2021. We raden alle klanten die nog steeds gebruikmaken van oude API's aan om binnenkort over te stappen op de extra voordelen van Resource Manager, waaronder resourcegroepering, tags, een gestroomlijnd implementatie- en beheerproces en fijnkeurig toegangsbeheer met behulp van op rollen gebaseerd toegangsbeheer van Azure (Azure RBAC).

Zie de [TechNet-blog](https://blogs.technet.microsoft.com/meamcs/2016/12/22/difference-between-azure-service-manager-and-azure-resource-manager/)Azure Resource Manager meer informatie Service Manager over azure-Service Manager.

Voor meer informatie over Service Manager en Resource Manager API's voor Azure Service Bus, Relay en Event Hubs, zie onze REST API documentatie:

- [Azure Service Bus](/rest/api/servicebus/)
- [Azure Event Hubs](/rest/api/eventhub/)
- [Azure Relay](/rest/api/relay/)
