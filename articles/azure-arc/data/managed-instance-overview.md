# <a name="azure-arc-enabled-sql-managed-instance-overview"></a>Overzicht van SQL Managed Instance met Azure Arc

SQL Managed Instance met Azure Arc is een Azure SQL-gegevensservice die kan worden aangemaakt op de infrastructuur van uw keuze. 

[!INCLUDE [azure-arc-data-preview](../../../includes/azure-arc-data-preview.md)]

## <a name="description"></a>Beschrijving

SQL Managed Instance met Azure Arc is bijna 100% compatibel met de nieuwste SQL Server-database-engine, en maakt het bestaande klanten met SQL Server mogelijk hun toepassingen over te brengen naar Azure Arc-gegevensservices met minimale wijzigingen van hun toepassingen en databases, terwijl de gegevenssoevereiniteit behouden blijft. Tegelijkertijd bevat SQL Managed Instance ingebouwde beheermogelijkheden die de beheer-overhead drastisch verminderen.
