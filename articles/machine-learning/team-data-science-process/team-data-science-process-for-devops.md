# <a name="team-data-science-process-for-developer-operations"></a>Team data Science process voor ontwikkelaars bewerkingen

In dit artikel worden de functies voor ontwikkel aars (DevOps) besproken die specifiek zijn voor een geavanceerde analyse en implementatie van Cognitive Services oplossingen. Deze trainings materialen implementeren het team data Science process (TDSP) en micro soft en open source software en tool kits, handig voor het voors telling, uitvoeren en leveren van data Science-oplossingen. Er wordt verwezen naar onderwerpen over de DevOps-hulpprogramma keten die specifiek zijn voor data Science-en AI-projecten en-oplossingen.
