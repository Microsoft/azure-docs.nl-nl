# <a name="generate-arm-template-for-cloud-services-extended-support-using-the-azure-portal"></a>ARM-sjabloon genereren voor Cloud Services (uitgebreide ondersteuning) met behulp van de Azure Portal

In dit artikel wordt uitgelegd hoe u de ARM-sjabloon en het parameter bestand kunt ophalen uit de [Azure Portal](https://portal.azure.com) nadat de Cloud service is ge√Ømplementeerd (uitgebreide ondersteuning). De ARM-sjabloon en het parameter bestand kunnen worden gebruikt in toekomstige implementaties om een Cloud service bij te werken of bij te werken (uitgebreide ondersteuning)

## <a name="get-arm-template-via-portal"></a>ARM-sjabloon ophalen via de portal

  1. Ga naar de resource groep en selecteer implementaties.
  :::image type="content" source="media/generate-template-portal-1.png" alt-text="Afbeelding toont het selecteren van implementaties onder resource groep op het Azure Portal.":::
  
  2. Selecteer uw Cloud service (uitgebreide ondersteuning) en klik op sjabloon.
  :::image type="content" source="media/generate-template-portal-2.png" alt-text="Afbeelding toont het selecteren van sjabloon onder Cloud service (uitgebreide ondersteuning) op de Azure Portal.":::
  
  3. Down load uw sjabloon-en parameter bestanden. Deze kunnen worden gebruikt voor toekomstige implementaties via Power shell.
  :::image type="content" source="media/generate-template-portal-3.png" alt-text="Afbeelding toont het downloaden van sjabloon bestand op het Azure Portal.":::
