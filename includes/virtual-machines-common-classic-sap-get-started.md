## <a name="planning-and-implementation"></a>Planning en implementatie
Titel: SAP NetWeaver op Azure Virtual Machines – plannings-en implementatie handleiding

Samenvatting: dit is het document waarmee u begint als u overweegt SAP NetWeaver te gebruiken op virtuele Azure-machines. Op basis van deze plannings- en implementatiehandleiding kunt u bepalen of een bestaand of gepland SAP NetWeaver-systeem kan worden geïmplementeerd in een omgeving met virtuele Azure-machines. Hierin komen verschillende SAP NetWeaver-implementatiescenario’s aan bod en worden specifieke SAP-configuraties voor Azure beschreven. Het document bevat alle configuratiegegevens die u voor SAP/Azure nodig hebt om een hybride SAP-landschap uit te voeren. Daarnaast wordt beschreven welke maatregelen u kunt nemen om een hoge beschikbaarheid van SAP NetWeaver-systemen in IaaS te garanderen.

## <a name="sap-dbms-on-azure"></a>SAP DBMS op Azure
Titel: SAP DBMS in azure Deployment Guide (Engelstalig)

Samenvatting: dit document bevat plannings- en implementatie-informatie voor de DBMS-systemen die moeten worden uitgevoerd in combinatie met SAP. Het eerste deel bestaat uit algemene overwegingen. De volgende delen van het artikel hebben betrekking op implementaties van verschillende DBMS-systemen in Azure die worden ondersteund door SAP. Verschillende DBMS-weer gegeven zijn SQL Server, SAP ASE, Oracle, SAP MaxDB en IBM DB2 voor Linux, UNIX en Windows. Er wordt aangegeven waarmee u rekening moet houden wanneer u SAP-systemen in Azure uitvoert in combinatie met deze DBMS-systemen. Er wordt onder andere beschreven welke back-ups en methoden voor hoge beschikbaarheid door de verschillende DBMS-systemen in Azure worden ondersteund voor gebruik met SAP-toepassingen.

Bijgewerkt: december 2015

## <a name="sap-netweaver-on-azure"></a>SAP net-Weaver op Azure
Titel: SAP NetWeaver: een op Azure gebaseerde oplossing voor herstel na nood gevallen bouwen

Samen vatting: dit document bevat stapsgewijze instructies voor het bouwen van een Azure-oplossing voor herstel na nood geval voor SAP net-Weaver. In de beschreven oplossing wordt ervan uitgegaan dat het SAP-landschap op basis van Hyper-V op de virtuele machines wordt uitgevoerd. In het eerste deel van de document Azure Site Recovery (ASR)-services worden geïntroduceerd in de bijbehorende onderdelen. In het tweede gedeelte van het document worden specifieke Details voor de op basis van SAP net-Weaver gebaseerde landschappen beschreven. Mogelijkheden van het gebruik van ASR met SAP NetWeaver-toepassings exemplaren en SAP Central-Services worden besproken en beschreven. De focus van het tweede deel is het gebruik van ASR voor SAP Central-Services die zijn beveiligd met Windows Server failover cluster-configuraties.

Bijgewerkt: september 2015
