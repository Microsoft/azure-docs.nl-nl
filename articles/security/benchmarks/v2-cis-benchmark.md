# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Toewijzing van Azure Security Bench Mark v2 en CIS Microsoft Azure Stichtings basis benchmark

Het DIS Microsoft Azure Stichtings benchmark is de beveiligings richtlijnen die door Center voor Internet beveiliging worden verschaft voor het tot stand brengen van een veilige basislijn configuratie voor Azure. Het bereik van de Bench Mark is het instellen van het basis niveau van beveiliging bij het aannemen van Azure-Cloud. Deze bench Mark wordt uitgelijnd met de [Azure Security Bench Mark v2](overview.md).

We bieden een toewijzing tussen de [meest recente 1.3.0 van de basis versie v van CIS Microsoft Azure](https://www.cisecurity.org/benchmark/azure/) en [Azure Security Bench Mark v2](overview.md). Dit helpt u te begrijpen hoe de aanbevelingen voor beide benchmarks worden uitgelijnd. U kunt de volledige toewijzing [hier](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)downloaden.
