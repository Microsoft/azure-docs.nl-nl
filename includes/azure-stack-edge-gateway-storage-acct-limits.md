Voor de meest actuele informatie over deze limieten gaat u naar [Azure Blob-opslag schaal doelen](../articles/storage/common/scalability-targets-standard-account.md#scale-targets-for-standard-storage-accounts) en [Azure files schaal doelen](../articles/storage/common/scalability-targets-standard-account.md#scale-targets-for-standard-storage-accounts).

|---------------------------------------------------------------------|------------------------|
