# <a name="how-to-find-your-azure-tenant-id"></a>Uw Azure-Tenant-ID zoeken

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>Uw Tenant-ID vinden in de portal

1. Meld u aan bij [Azure Portal](https://portal.azure.com).
1. Selecteer het Azure Active Directory pictogram onder Azure-Services. De Active Directory-pagina wordt geopend.
1. Uw Tenant-ID bevindt zich onder **basis informatie**.
1. Kopieer de Tenant-ID en plak deze in een tekst-document van uw keuze voor later gebruik.
