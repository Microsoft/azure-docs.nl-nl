# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Aan de slag met het ontwikkelen van ingesloten Azure IoT-apparaten

**Van toepassing op**: [Ontwikkeling van ingesloten apparaten](about-iot-develop.md#embedded-device-development)

Deze aan de slag-handleiding bevat een aantal quickstarts die laten zien hoe u aan de slag kunt gaan met ingesloten apparaten en Azure IoT. 

In elke quickstart voltooit u de volgende basistaken:
* Een set ingesloten ontwikkelprogramma's installeren voor het programmeren van een specifiek apparaat in C
* Bouw een afbeelding met Azure RTOS onderdelen en voorbeelden en flash vervolgens een apparaat
* Een apparaat veilig verbinden met Azure IoT
* Telemetrie van apparaten weergeven, eigenschappen weergeven en cloud-naar-apparaat-methoden aanroepen
