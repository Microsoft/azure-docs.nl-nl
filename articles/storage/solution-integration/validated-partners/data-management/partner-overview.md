# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Azure Storage Data Governance-, beheer-en migratie partners

In dit artikel worden micro soft-partner bedrijven uitgelegd die zijn geïntegreerd met Azure Storage waarmee u de algehele mogelijkheden voor gegevens beheer kunt verbeteren. Deze partner oplossingen kunnen ondersteuning bieden voor opslag analyse en rapportage, neutraal migratie, replicatie, Cloud lagen en data governance.
