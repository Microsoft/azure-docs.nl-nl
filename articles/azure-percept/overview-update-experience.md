# <a name="the-azure-percept-dk-update-experience"></a>De update-ervaring voor Azure percept DK

Met Azure percept DK hebt u twee opties om het besturings systeem en de firmware van uw dev kit bij te werken: over-the-Air (OTA) of via USB. Het bijwerken van OTA is een eenvoudige manier om apparaten up-to-date te houden, terwijl het bijwerken van USB een goede optie is als OTA niet mogelijk is of wanneer u de fabrieks instellingen van uw apparaat wilt herstellen. Om ervoor te zorgen dat u kunt profiteren van welke update methode het beste voor u is, is er een verzameling instructies voor het samen stellen van uw hulp. 

- [Azure IoT Hub instellen voor het implementeren van de Air-updates voor uw Azure percept DK](./how-to-set-up-over-the-air-updates.md)
- [Uw Azure percept DK bijwerken via de lucht](./how-to-update-over-the-air.md)
- [Uw Azure percept DK via USB bijwerken](./how-to-update-via-usb.md)
