# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>Azure Storage partners voor de primaire en secundaire opslag

In dit artikel worden micro soft-partner bedrijven uitgelegd die een NAS-oplossing (Network Attached Storage) of een Storage Area Network (SAN) leveren. De oplossing kan on-premises, in azure of een hybride oplossing zijn die gebruikmaakt van Azure Storage als een kosten efficiënte laag. Deze oplossingen kunnen klanten in staat stellen om dezelfde oplossing te gebruiken in een van hun omgevingen.
