# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Aan de slag met het ontwikkelen van Azure IoT embedded-apparaten

**Van toepassing op**: [ontwikkelen van Inge sloten apparaten](about-iot-develop.md#embedded-device-development)

Deze hand leiding bevat een aantal Quick starts waarin u kunt zien hoe u kunt werken met Inge sloten apparaten en Azure IoT. 

In elke Snelstartgids voert u de volgende basis taken uit:
* Een set Inge sloten ontwikkel hulpprogramma's installeren voor het Program meren van een specifiek apparaat in C
* Bouw een installatie kopie die onderdelen en voor beelden van Azure RTO'S bevat, en knip vervolgens een apparaat
* Een apparaat veilig verbinden met Azure IoT
* Telemetrie van apparaten weer geven, eigenschappen weer geven en Cloud-naar-apparaat-methoden aanroepen
