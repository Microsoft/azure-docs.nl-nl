# <a name="agent-portfolio-overview-and-os-support-preview"></a>Overzicht van agent portfolio en ondersteuning van besturings systemen (preview-versie)

Azure Defender voor IoT biedt een grote Port Folio met agents die zijn gebaseerd op het apparaattype. 

## <a name="standalone-agent"></a>Zelfstandige agent

De zelfstandige agent heeft betrekking op de meeste Linux-besturings systemen, die kunnen worden geïmplementeerd als een binair pakket of als bron code die kan worden opgenomen als onderdeel van de firmware en aanpassing en aanpassing toestaan op basis van de behoeften van de klant. Een voor beeld van een besturingssysteem ondersteuning: 

| Besturingssysteem | AMD64 | ARM32v7 |
|--|--|--|
| Debian 9 | ✓ | ✓ |
| Ubuntu 18.04 | ✓ |  |
| Ubuntu 20.04 | ✓ |  |

Neem contact op met uw account manager of stuur een e-mail naar voor meer informatie over de ondersteuning van besturings systemen, of om toegang tot de bron code aan te vragen, zodat u deze kunt opnemen als onderdeel van de firmware van het apparaat <defender_micro_agent@microsoft.com> . 

## <a name="azure-rtos-micro-agent"></a>Azure RTO'S micro agent

De Azure Defender voor IoT micro agent biedt een uitgebreide en lichte beveiligings oplossing voor apparaten die gebruikmaken van Azure RTO'S. Azure Defender voor IoT micro agent voorziet in dekking voor veelvoorkomende bedreigingen en mogelijke schadelijke activiteiten op RTO'S-apparaten (realtime Operating System). De micro agent is ingebouwd als onderdeel van het onderdeel Azure RTO'S NetX Duo en bewaakt de netwerk activiteit van het apparaat. 

De Azure Defender voor IoT micro agent is ingebouwd als onderdeel van het Azure RTO'S NetX Duo-onderdeel en bewaakt de netwerk activiteit van het apparaat. De micro agent bestaat uit een uitgebreide en lichte beveiligings oplossing die een dekking biedt voor veelvoorkomende bedreigingen en mogelijke schadelijke activiteiten op een real-time besturings systeem (RTO'S).
