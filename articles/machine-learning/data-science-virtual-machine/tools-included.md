# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Welke hulpmiddelen worden meegeleverd met de Azure Data Science Virtual Machine?

De Data Science Virtual Machine is een eenvoudige manier om gegevens te verkennen en machine learning in de Cloud uit te voeren. De data Science Virtual Machines zijn vooraf geconfigureerd met het volledige besturings systeem, beveiligings patches, stuur Programma's en populaire software voor gegevens wetenschap en ontwikkeling. U kunt de hardware-omgeving kiezen, variërend van goedkope CPU-georiënteerde computers tot zeer krachtige computers met meerdere Gpu's, NVMe-opslag en grote hoeveel heden geheugen. Voor computers met Gpu's worden alle Stuur Programma's geïnstalleerd, alle machine learning frameworks zijn de versie die overeenkomt met de GPU-compatibiliteit en de versnelling is ingeschakeld in alle toepassings software die Gpu's ondersteunt.

Het Data Science Virtual Machine wordt geleverd met de meest nuttige hulpprogram ma's voor gegevens wetenschap vooraf geïnstalleerd.

## <a name="how-to-choosecondaenvironment"></a>Conda-omgeving kiezen  

> [!NOTE]
> Er zijn belang rijke wijzigingen wanneer u overschakelt naar de nieuwe versie van Data Science Virtual Machine van de oude versie. 

**Overschakelen van Windows Server 2016 naar Windows Server 2019** </br> 

* De Conda-omgeving is ingesteld op python37_default. Python2 wordt niet ondersteund. Als u tensor flow of pytorch in uw python Conda-omgeving gebruikt, gebruikt u py37_tensorflow of py37_pytorch respectievelijk. Als u alleen AzureML-gebruik gebruikt azureml_py36_automl </br>

* Als u tensor flow of pytorch in uw Conda-omgeving met AzureML gebruikt, gebruikt u azureml_py36_tensorflow of azureml_py36_pytorch respectievelijk. </br>

**Overschakelen van Ubuntu 16,04 naar Ubuntu 18,04** </br>

* Als u tensor flow of pytorch in uw azureml_py36 Conda-omgeving gebruikt, azureml_py36_tensorflow of azureml_py36_pytorch respectievelijk. </br>

* Gebruik py37_tensorflow of py37_pytorch als u tensor flow of pytorch in uw py37 Conda-omgeving gebruikt. </br>

* Anders gebruikt u python37_default of gebruikt u alleen azureml_py36 gebruik azureml_py36_automl </br>
