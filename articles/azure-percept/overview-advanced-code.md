# <a name="advanced-development-with-azure-percept"></a>Geavanceerde ontwikkeling met Azure percept
Met Azure percept kunnen software ontwikkelaars en gegevens wetenschappers gebruikmaken van geavanceerde code werk stromen voor het beheer van de AI-levens cyclus. Dankzij een groeiende open-source bibliotheek kunnen ze voor beelden gebruiken om aan de slag te gaan met hun AI-ontwikkelings reis en productie klare oplossingen te bouwen.

## <a name="get-started-with-the-advanced-development-tutorials"></a>Aan de slag met de zelf studies voor geavanceerde ontwikkel aars

Meer informatie over alle beschik bare [Azure PERCEPT AI-modellen](./overview-ai-models.md).

Raadpleeg [Azure PERCEPT DK Advanced Development github](https://github.com/microsoft/azure-percept-advanced-development) voor actuele hulp, zelf studies en voor beelden voor zaken als:

* Een aangepast AI-model naar het apparaat brengen
* Een model bijwerken dat al wordt ondersteund met overboeking Learning
* En meer

## <a name="next-steps"></a>Volgende stappen

Meer informatie over alle beschik bare [Azure PERCEPT AI-modellen](./overview-ai-models.md). Als geen van deze modellen aan uw behoeften voldoet, kunt u het geavanceerde code traject gebruiken om uw eigen model of computer vision-pijp lijn naar de percept DK te brengen. Als u denkt dat u anderen zou helpen, kunt u ook een pull-aanvraag openen.
