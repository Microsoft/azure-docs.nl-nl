# <a name="configure-an-openvpn-client-for-azure-virtual-wan"></a>Een OpenVPN-Client voor Azure Virtual WAN configureren

Dit artikel helpt u bij het configureren van **openvpn- &reg; protocol** -clients. U kunt ook de Azure VPN-client voor Windows 10 gebruiken om verbinding te maken via het OpenVPN-protocol. Meer instructies vindt u [hier](openvpn-azure-ad-client.md)

## <a name="before-you-begin"></a>Voordat u begint

Een VPN-configuratie (punt-naar-site) maken voor een gebruiker. Zorg ervoor dat u "OpenVPN" voor het tunnel type selecteert. Zie [een P2S-configuratie maken voor Azure Virtual WAN](virtual-wan-point-to-site-portal.md#p2sconfig)voor instructies.
