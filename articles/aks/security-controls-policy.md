# <a name="azure-policy-regulatory-compliance-controls-for-azure-kubernetes-service-aks"></a>Besturingselementen voor Naleving van Azure Policy-regelgeving voor AKS (Azure Kubernetes Service)

[Naleving van regelgeving in Azure Policy](../governance/policy/concepts/regulatory-compliance.md) biedt door Microsoft gemaakte en beheerde initiatiefdefinities, bekend als _ingebouwde modules_, voor de **nalevingsdomeinen** en **beveiligingsmaatregelen** met betrekking tot verschillende nalevingsstandaarden. Op deze pagina worden de **nalevingsdomeinen** en **beveiligingsmaatregelen** voor AKS (Azure Kubernetes Service) weergegeven.
U kunt de ingebouwde modules voor een **beveiligingsmaatregel** afzonderlijk toewijzen om uw Azure-resources de specifieke standaard te laten naleven.
