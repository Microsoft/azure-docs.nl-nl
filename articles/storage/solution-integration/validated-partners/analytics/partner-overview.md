# <a name="azure-storage-analytics-and-big-data-partners"></a>Azure Opslaganalyse-en Big Data-partners

In dit artikel worden micro soft-partner bedrijven uitgelegd die zijn geïntegreerd met Azure Data Lake Storage. Ze omvatten werk belastingen zoals moderne Data Warehouse, geavanceerde analyses, realtime analyse, enzovoort. Deze partners maken gebruik van de hiërarchische naam ruimte in Azure Storage om de oplossing te optimaliseren en efficiënt uit te voeren in Azure.
