## <a name="install-the-azure-functions-extension"></a>De Azure Function-extensie installeren

U kunt de extensie Azure Functions gebruiken om functies te maken en te testen en te implementeren in Azure.

1. Open in Visual Studio Code de optie **Extensies** en zoek naar **Azure-functies** of open deze koppeling in Visual Studio Code`vscode:extension/ms-azuretools.vscode-azurefunctions`.

1. Selecteer **Installeren** om de extensie in Visual Studio Code te installeren:

    ![De extensie voor Azure Functions installeren](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. Selecteer het pictogram Azure op de activiteitenbalk na de installatie. Het gebied Azure Functions moet nu in de zijbalk worden weergegeven.

    ![Gebied Azure Functions in de zijbalk](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)
