# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple for Cloud solutions provider-programma: veelgestelde vragen


## <a name="overview"></a>Overzicht
Hieronder vindt u vragen en antwoorden die u mogelijk als een StorSimple-partner hebt wanneer u een StorSimple-virtuele matrix implementeert of beheert in Azure Portal.

Vragen en antwoorden zijn ingedeeld in de volgende categorieën:

 - [Model van de StorSimple en het Cloud solutions provider-programma (CSP)](#storsimple-and-csp-model)
 - [Virtuele array implementeren en beheren als een partner](#deploy-and-manage-storsimple-as-a-partner)
 - Ondersteuning voor StorSimple-partners voor CSP

## <a name="storsimple-and-csp-model"></a>StorSimple en CSP-model

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Wat betekent het dat StorSimple als onderdeel van het CSP-programma (Cloud solutions provider)?
 Met StorSimple voor CSP kunnen onze CSP-partners hun klanten StorSimple virtuele array verkopen. Met dit model kunnen de CSP-partners eigenaar zijn van de end-to-end levens cyclus van klanten met directe inrichting, facturering en ondersteuning van de Cloud Services van micro soft. 

Ga naar [StorSimple in CSP-programma](https://www.microsoft.com/en-us/server-cloud/products/storsimple/)voor meer informatie. 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-microsoft-365-and-enterprise-mobility-suite-ems"></a>Is StorSimple een op gebruik gebaseerd of op basis van Seat-licenties, zoals Microsoft 365 en Enter prise Mobility Suite (EMS)?
StorSimple is een op gebruik gebaseerde service. Klanten die StorSimple Virtual array aanschaffen via de CSP-route, betalen voor het gebruik van virtuele matrix en Azure, gefactureerd als afzonderlijke regel items in de CSP-factuur. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-microsoft-365-and-the-enterprise-mobility-suite"></a>Is het StorSimple-bedrijfs model vergelijkbaar met dat van andere services, zoals het Microsoft 365 en de Enter prise Mobility Suite? 
Ja. Het model is op dezelfde manier als andere op gebruik gebaseerde services in Azure. De groothandels korting voor StorSimple is vergelijkbaar met die van andere Azure-Services. Dit model biedt onze CSP-partners een mogelijkheid om meer te verkopen en Azure-verbruik via StorSimple Virtual array te stimuleren.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Welke StorSimple-Sku's zijn beschikbaar bij het starten?
Dezelfde Sku's die beschikbaar zijn voor de zelfstandige StorSimple Services. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Is er een verschil tussen het StorSimple dat rechtstreeks bij micro soft is gekocht onder EA en StorSimple als onderdeel van de CSP?
Nee. Onder CSP kunnen klanten ook andere services kopen die door de CSP-partner en StorSimple Virtual array (SVA) onder één factuur worden aangeboden. Houd er rekening mee dat klanten het fysieke apparaat van de StorSimple 8000-serie willen inkopen, maar nog steeds de EA-route moeten door lopen. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>In welke markten is StorSimple beschikbaar voor CSP bij het starten?
StorSimple Virtual array voor CSP is beschikbaar in alle markten waar StorSimple vandaag beschikbaar is. Ga voor meer informatie naar de [lijst met regio's waar StorSimple beschikbaar is](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Wat voor soort oplossingen kan een partner leveren met StorSimple en CSP? 
Er zijn meerdere oplossingen:

- De partners kunnen StorSimple Virtual array verkopen. 
- De partners kunnen oplossingen bieden die zijn gebouwd rond StorSimple en andere Azure-Services of software van derden. Bekijk voor meer informatie enkele mogelijke [oplossingen die van StorSimple zijn](./storsimple-disaster-recovery-using-azure-site-recovery.md).
- Ze kunnen ook nieuwe bedrijfs modellen ontdekken door beheerde services te leveren.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Wat zijn de prikkels die beschikbaar zijn voor StorSimple CSP-partners? Kom ik in aanmerking voor de prikkel van een cryptografie programma?
Voor meer informatie over stimulansen gaat u naar het [cryptografie programma](https://partner.microsoft.com/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>StorSimple implementeren en beheren als een partner

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Hoe kan ik de virtuele StorSimple-matrix in CSP beheren? 
U kunt StorSimple Virtual array-abonnementen toevoegen aan uw klanten account via het partner centrum. Daarnaast kunt u de Azure Portal gebruiken om gebruikers toe te voegen aan de abonnementen.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Is de Azure Portal hetzelfde als andere services voor CSP? 
Ja. Met de virtuele StorSimple-matrix voor CSP kunt u de beste manier om partners toegang te geven tot de Azure Portal, via het [partner centrum](https://partnercenter.microsoft.com/) waar ze klanten en abonnementen kunnen beheren. Voor het beheren van StorSimple-abonnementen moet de partner of klant (afhankelijk van de verleende machtigingen) zich aanmelden bij de Azure Portal. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Vervoert micro soft een nieuwe portal voor StorSimple voor CSP? 
Nee. U kunt de virtuele StorSimple-matrix in CSP beheren via de Azure Portal.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Ik heb een zelfstandig StorSimple-abonnement ingericht voor mijn klant. Moet ik de Azure Portal gebruiken om mijn apparaat te beheren? 
Ja. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Wordt het micro soft-account team gecompenseerd als gevolg van de verkoop van StorSimple via CSP?
Ja. De verkoop Manager van de partner en het klant account team voor de eind klant worden gecompenseerd voor de omzet belasting van de leverancier. De partners worden naar verwachting de potentiële klant te nemen bij het verkopen van hun gedifferentieerde waarde en aanbiedingen.
