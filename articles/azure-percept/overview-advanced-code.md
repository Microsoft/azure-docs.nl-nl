# <a name="advanced-development-with-azure-percept"></a>Geavanceerde ontwikkeling met Azure percept

Met Azure percept kunnen software ontwikkelaars en gegevens wetenschappers gebruikmaken van geavanceerde code werk stromen voor het beheer van de AI-levens cyclus. Dankzij een groeiende open-source bibliotheek kunnen ze voor beelden gebruiken om aan de slag te gaan met hun AI-ontwikkelings reis en productie klare oplossingen te bouwen.

## <a name="get-started-with-advanced-development"></a>Aan de slag met geavanceerde ontwikkeling

Raadpleeg de [github van Azure PERCEPT DK Advanced Development](https://github.com/microsoft/azure-percept-advanced-development) voor up-to-date hulp, zelf studies en voor beelden voor zaken als:

- Een aangepast AI-model implementeren in uw Azure percept DK
- Een ondersteund model bijwerken met overboeking Learning
- En meer

## <a name="next-steps"></a>Volgende stappen

Meer informatie over de beschik bare [Azure PERCEPT AI-modellen](./overview-ai-models.md). Als geen van deze modellen aan uw behoeften voldoet, gebruikt u de geavanceerde code om uw eigen model of computer vision-pijp lijn naar de percept DK te brengen. Als u een bijdrage hebt die anderen kunnen helpen, kunt u ook gratis een pull-aanvraag openen.
