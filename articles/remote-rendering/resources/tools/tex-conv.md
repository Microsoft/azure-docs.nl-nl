# <a name="texconv---texture-conversion-tool"></a>TexConv-patroon conversie programma

Het hulp programma TexConv is een extern opdracht regel programma voor het uitvoeren van patroon conversies in batches.
Het hulp programma kan worden gevonden in deze git-opslag plaats: [github-opslag plaats voor het TexConv-hulp programma](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Voor externe rendering hoeven bitmappatronen niet te worden geconverteerd, omdat de conversie stap kan omgaan met algemene afbeeldings bestands indelingen en structuren converteert naar de juiste runtime-indeling. Dit hulp programma kan echter nuttig zijn om structuren lokaal te converteren met aangepaste instellingen.
